# Moje repozytorium dodatków Home Assistant

## Traccar Custom
Dodatek z najnowszą wersją serwera Traccar (6.11.1).

### Instalacja:
1. W Home Assistant: Ustawienia → Dodatki → Repozytoria → Dodaj repozytorium.
2. Wklej URL: `https://github.com/<twoje-repozytorium>`.
3. Przeładuj repozytoria i zainstaluj dodatek **Traccar Custom**.
